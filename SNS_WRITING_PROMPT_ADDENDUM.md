# 블루블랙 SNS 추가 필수 지침

이 문서는 기존 SNS 작성 마스터 프롬프트보다 우선 적용한다.

## 브랜드 공식 인스타그램 계정

1. 모든 제품 게시글에는 해당 브랜드의 공식 인스타그램 계정을 확인해 구매 안내 다음 줄, 해시태그 앞에 넣는다.
2. 브랜드 본사 계정과 국가별 계정이 모두 있을 경우 해당 제품의 출시 국가 또는 국내 유통과 가장 직접적으로 관련된 공식 계정을 우선한다.
3. 협업 제품은 브랜드 공식 계정 다음 줄에 확인된 작가·캐릭터·협업처 공식 계정을 추가할 수 있다.
4. 계정은 공식 홈페이지, 공식 프로필, 공식 수입원 또는 브랜드가 직접 연결한 채널로 확인한다.
5. 공식 여부를 확인하지 못한 계정은 추측해 태그하지 않는다. 내부 화면에 ‘공식 계정 확인 필요’로 남긴다.
6. 원고 5개 모두 같은 공식 계정 표기를 포함한다.

## 해시태그

1. 인스타그램 게시글의 해시태그는 기본 20개로 작성한다.
2. 제품에 따라 18~22개 범위에서 조정할 수 있으나 특별한 이유가 없으면 20개를 사용한다.
3. 다음 비율을 기준으로 구성한다.
   - 브랜드명·제품명·공식 영문명: 4~6개
   - 제품군·규격·핵심 기능: 5~7개
   - 사용 장면·문구 관심사: 4~6개
   - 블루블랙 펜샵·매장 관련: 2~4개
4. 관련 없는 인기 태그를 조회수 목적으로 넣지 않는다.
5. 같은 뜻의 한국어·영문 태그를 과도하게 반복하지 않는다.
6. 원고 5개는 핵심 태그를 공통으로 유지하되, 버전의 강조점에 맞춰 일부 태그를 다르게 구성한다.

## 사진 촬영·구도

1. 인스타그램 피드용 사진은 세로 4:5 비율을 우선한다.
2. 썸네일에서 제품명과 핵심 디자인이 잘리지 않도록 중앙 안전 영역을 확보한다.
3. 게시물당 최소 다음 6종의 사진을 준비한다.
   - 대표 정면 컷: 제품 전체와 핵심 디자인이 한눈에 보이는 사진
   - 전체 구성 탑뷰: 색상·옵션·패키지를 일정한 간격으로 배열한 사진
   - 디테일 컷: 로고, 규격, 라벨, 닙, 패턴 또는 구성품 확대 사진
   - 사용 장면 컷: 실제 필기, 충전, 조립, 꾸미기 등 손이 들어간 사진
   - 비교 컷: 색상·굵기·필기 농도·크기를 동일 조건에서 비교한 사진
   - 정보 확인 컷: 패키지 뒷면, 구성품, 주요 표기를 확인할 수 있는 사진
4. 메인 제품은 화면 면적의 약 60~75%를 차지하도록 배치한다.
5. 소품은 제품보다 눈에 띄지 않게 하고 실제 판매 구성품으로 오해되지 않도록 한다.
6. 여러 제품을 배열할 때 수직·수평과 간격을 일정하게 맞춘다.
7. 플라스틱과 금속의 반사를 줄이기 위해 조명을 정면이 아닌 약 45도 측면에서 부드럽게 비춘다.
8. 필기 비교는 같은 샤프, 같은 종이, 같은 압력, 같은 조명에서 진행한다.
9. AI 시안은 구도 참고용으로만 사용한다. 시안 속 글자, 규격, 패키지 디자인은 실제 제품 정보로 사용하지 않는다.
10. 촬영 후 대표·구성·디테일·사용·비교·정보 확인 컷이 모두 확보됐는지 확인한다.

## 네옥스 그래파이트 WAKARU 촬영 순서

1. 12종 전체를 6개씩 두 줄로 정렬한 밝은 배경 탑뷰
2. 패키지 앞면과 뒷면을 나란히 놓은 규격 확인 컷
3. 동봉 스티커와 꾸미기 전 케이스를 함께 배치한 구성 컷
4. 손으로 스티커를 붙이는 순간을 촬영한 사용 장면
5. 꾸미기 전과 후를 같은 위치에서 촬영한 비교 컷
6. 기존 네옥스 심과 동일한 샤프로 쓴 농도·부드러움 비교 컷
7. 지우개 사용 전후와 잔흔을 가까이 촬영한 리뷰 컷
8. 샤프, 노트, 완성된 케이스를 함께 놓은 최종 감성 컷
